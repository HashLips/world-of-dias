# Veloria ring clearance rule

## Name

Veloria ring clearance rule

## Overview

Movement between Veloria rings is permission-graded by role, timing, and system load.

## Description

Inner-ring access requires matching clearance to purpose. During stress windows, clearance narrows automatically and nonessential crossings are deferred.

## Effect

- Protects core infrastructure from chaotic surges.
- Reinforces perceptions of classed or opaque governance.

## Notes

The rule is framed as safety engineering by officials and as social filtering by critics.
