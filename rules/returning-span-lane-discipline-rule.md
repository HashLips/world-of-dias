# Returning Span lane discipline rule

## Name

Returning Span lane discipline rule

## Overview

Crossing the Returning Span requires declared lane choice and pass logging.

## Description

Travelers must commit to first, second, or third lane behavior profiles before entry. Unlogged lane switching during drift windows is treated as high-risk negligence.

## Effect

- Increases repeat-route reliability.
- Provides structured data for Refrain Walker forecasts.

## Notes

"Unmarked lane change" is a common root cause in Span incident reports.
