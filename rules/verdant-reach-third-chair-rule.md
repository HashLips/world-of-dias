# Verdant Reach third-chair rule

## Name

Verdant Reach third-chair rule

## Overview

Formal mediation meals in Verdant Reach are set with one unassigned seat.

## Description

The unassigned chair represents return possibility for absent stakeholders. Agreements finalized without offering this seat can be contested as procedurally incomplete.

## Effect

- Keeps disputes from hardening into permanent exclusion.
- Embeds reconciliation logic into ordinary routine.

## Notes

The rule is practical, not sentimental: it creates recurring opportunity windows.
