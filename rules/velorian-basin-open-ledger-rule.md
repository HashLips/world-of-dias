# Velorian Basin open-ledger rule

## Name

Velorian Basin open-ledger rule

## Overview

Basin trade disputes are resolved in visible records before private settlements are accepted.

## Description

Parties may negotiate privately, but final obligations must be logged in shared ledger formats available to ward mediators. Hidden clauses are unenforceable in civic courts.

## Effect

- Lowers fraud repeat rates.
- Encourages informal trust circles to interface with formal process.

## Notes

This rule helps explain why Basin commerce remains high-volume without constant collapse.
