# Unfinished persistence scale rule

## Name

Unfinished persistence scale rule

## Overview

Map claims in The Unfinished must include persistence class, not just location.

## Description

Routes and structures are tagged as fleeting, unstable, holding, steady, or enduring based on repeated pass confirmation. Position without persistence is treated as insufficient navigation data.

## Effect

- Enables useful movement in revising terrain.
- Prevents false certainty from single-pass maps.

## Notes

This rule underpins Draftkin map pedagogy across scaffold sectors.
