# Ashen Hearthline rest viability rule

## Name

Ashen Hearthline rest viability rule

## Overview

Travel plans on the Hearthline are valid only with mapped viable rest intervals.

## Description

Route approvals require current heat, shelter, and relay checks at declared stop points. Unsupported long pushes are treated as unsafe misconduct.

## Effect

- Prevents avoidable exhaustion loss in recovery terrain.
- Keeps Ember Keeper maintenance data central to movement planning.

## Notes

"No rest plan, no route" is a standard Hearthline enforcement phrase.
