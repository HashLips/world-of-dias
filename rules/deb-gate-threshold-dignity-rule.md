# DEB Gate threshold dignity rule

## Name

DEB Gate threshold dignity rule

## Overview

Gate force is lawful only when proportionate, documented, and non-humiliating to compliant entrants.

## Description

Ward officers may deny access and detain for cause, but performative degradation of lawful travelers is punishable. Command urgency does not nullify procedure logs.

## Effect

- Preserves legitimacy of strict gate operations.
- Creates traceable accountability in high-tension entry events.

## Notes

Frequently taught as "order without spectacle."
