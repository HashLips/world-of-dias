# Sketchy Gallery open curation rule

## Name

Sketchy Gallery open curation rule

## Overview

No single patron line may control consecutive major exhibitions beyond one cycle.

## Description

Curatorial authority rotates through declared windows with mandatory public review notes. Patron funding is allowed, but sequence capture is prohibited.

## Effect

- Preserves the gallery as a civic-cultural forum instead of a private brand arm.
- Produces constant argument over standards, which is considered healthy.

## Notes

The rule protects plurality while preserving Zephan Quell's role as final cycle coordinator.
