# Four-Orb Convergence

## Overview

Four-Orb Convergence is a reported Bloomline event where four eye-like floating spheres appear near high-pressure confluence zones shortly before major civic decisions.

## Description

Witnesses report coordinated hovering behavior, synchronized orientation shifts, and abrupt disappearance after stabilization intervals. The event has no confirmed species attribution.

## Conditions

Most reports cluster around heavy-current overlap periods, especially when Tideheart-led holding intervals precede contract or mediation outcomes.

## Cultural Interpretations

Some communities read the orbs as neutral witnesses. Others treat them as pressure markers indicating that collective tone has not yet settled.

## Notes

The artwork *Dots of Color* is commonly used as the primary visual reference for this phenomenon.
