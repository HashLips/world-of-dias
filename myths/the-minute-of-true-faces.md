# The Minute of True Faces

## Summary

A persistent F610 myth says each cycle contains one minute when all reflective surfaces show not appearance, but structural truth.

## Myth Text

Versions differ on what "true face" means: intention, history, or hidden fracture lines. The warning is consistent: clarity without preparation can harm as easily as it can heal.

## Notes

Used to justify both rigorous recording discipline and careful release of reveal-phase findings.
