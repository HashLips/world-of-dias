# Red Light (Alwen Rusk Portrait)

## Overview

*Red Light* is the core portrait reference for Alwen Rusk.

## Description

The image frames Alwen as a stern but restorative figure, emphasizing alert responsibility over spectacle.

Asset reference: `assets/red-light.jpg`.

## Real-World Role

Character artwork anchor for Hearthvale's second-beginning leadership model.

## Lore Connection

Frequently associated with cooperative labor systems and re-entry settlements in west Sorel.

## Notes

Used as primary visual shorthand for Alwen's restorative authority.
