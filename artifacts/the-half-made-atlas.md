# The Half-Made Atlas

## Overview

**The Half-Made Atlas** is the most referenced cartographic artifact associated with F960: a codex whose mapped layers appear stable in core regions while margin details revise between readings.

## Description

The atlas is physically ordinary in binding and scale but unusual in behavior. Repeated readers report that anchor sectors (Scaffold Arc, Margin Flats, Ghost Quarter) remain recognizable, while edge geometry and route annotations update in subtle, nonrandom ways.

It does not replace fieldcraft: users still annotate manually. Its value lies in revealing where revision pressure is currently high.

## Cultural Meaning

Within The Open Hand of Making, the atlas is treated as proof that maps can be authoritative without pretending to be final.

## Usage

Used as a baseline reference for travel planning, apprenticeship exercises, and dispute mediation when two maps disagree but both appear partially valid.

## Notes

Do not frame it as omniscient. The atlas is an aid to judgment, not a perfect oracle.
