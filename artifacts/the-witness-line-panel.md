# The Witness-Line Panel

## Overview

The **Witness-Line Panel** is a narrow carved board used in parts of The Unfinished to train observers to report what can be seen in one stable pass, without speculative fill.

## Description

The panel contains layered grooves that resolve into a face only from specific angles. Apprentices must describe the form using contour order rather than naming assumptions. Instructors say the method prevents "story drift" during unstable visibility.

## Cultural Meaning

In Draftkin teaching circles, the panel symbolizes restraint in testimony: accuracy before interpretation, line before legend.

## Usage

Used in witness training, conflict debriefs, and workshop rites where multi-pass disagreements need a shared baseline.

## Notes

Often referenced with the artwork **And This Is** as a paired visual doctrine: narrowed sight, disciplined report.
