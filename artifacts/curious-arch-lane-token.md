# Curious Arch Lane Token

## Overview

The Curious Arch Lane Token is a personal crossing token marked with the Curious Arch symbol and used to formalize lane commitment before Span entry.

## Description

Tokens are small, durable pieces carried on cord or clasp, each scored with lane mark and crossing cycle. The arch-and-sphere motif is etched on one side to indicate commitment through disciplined inquiry.

## Cultural Meaning

Carrying the token signals that a traveler has chosen a lane profile intentionally and accepts behavioral constraints for that crossing.

## Usage

Issued or validated at approach points, then checked at lane gates during high-drift periods. Unmarked crossings are allowed only under emergency escort.

## Notes

Frequently depicted in sacred-orientation artworks and Span rite teaching material.
